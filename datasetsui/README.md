# iAODE Dataset Browser

Web-based interface for browsing and exploring single-cell datasets from NCBI GEO.

## Features

- **Dataset Catalog**: Browse curated scRNA-seq and scATAC-seq datasets
- **GSE Studies**: View grouped datasets by GSE accession
- **Statistics**: Explore dataset statistics and distributions
- **Dark Mode**: Supports light and dark themes

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dataset browser.

## Part of iAODE

This project is part of the [iAODE](https://github.com/PeterPonyu/iAODE) framework for interpretable single-cell omics analysis.
